# Basic English Quiz

A simple Python console app to test your English skills.  
It asks 3 random questions, calculates your score, and shows your percentage of correct answers.

---

##  Features
-  3 grammar or vocabulary questions  
-  User input and result summary  
-  Percentage calculation  
-  Beginner-friendly logic

---

## Tech Stack
- **Language:** Python 3  
- **Concepts:** input/output, conditionals, loops, lists  

---

## How It Works
1. The program asks 3 questions.
2. You enter answers.
3. It calculates your score and displays your success rate.

Example:
